# Anthony Mora | R&D Software Engineer Portfolio

> "Bridging the gap between Industrial R&D and Modern Web Engineering."

This portfolio showcases my transition from 13+ years in Medical Device R&D to High-Performance Software Engineering. Built with a "Swiss Minimalist" aesthetic, it emphasizes precision, typography, and performance.
